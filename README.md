# mijia-web
自制米家网页版，仅支持控制灯具相关设备
## 使用
本项目通过Python开发，使用前请确保已安装Python
### 首次使用
首次使用需要安装依赖
```bush
pip install -r requirements.txt
```
### 登录
安装依赖后双击打开app.py，确认加载完成后打开`http://127.0.0.1:5000`,打开后会自动跳转至`http://127.0.0.1:5000/login`。请使用米家APP扫码登录。

首次登录完成后认证信息会存储至` ~/.config/mijia-api/auth.json`,首次登录后短时间内无需再次登录。
### 基础使用
本项目仅支持控制灯具相关设备，登录后即可控制。如果你有其他想法，欢迎PR或Issues。
